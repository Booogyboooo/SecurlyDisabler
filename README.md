# SecurlyPartialDisabler
Disable securlys website blocking features

## Prerequisites
You need access to Dev Tools (ctrl + shift + i)

## Usage
1. Navigate to chrome://serviceworker-internals/ <br>
2. Using control + f search for "securly.min.js" <br>
3. When found press the inspect button <br>
4. In the newly opened window paste in the script and press enter

## Info
This script does not remove securly or fully disable it. It only stops securly from blocking websites. You need to do this every time you log into the chromebook.
